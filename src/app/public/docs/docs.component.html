String Analyzer

The String Analyzer is a tool designed to analyze a given word. Enter the word you wish to analyze and, if needed, select from autocomplete suggestions. Upon searching, the tool displays the length of the longest substring, along with a list of all substrings, highlighting each longest substring found.
