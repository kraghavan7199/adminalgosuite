AlgoSuite Documentation

AlgoSuite is a powerful tool for algorithm visualization and string analysis, featuring two main modules: String Explorer and Tree Visualizer.

String Explorer

The String Explorer module provides comprehensive substring analysis capabilities.

Features

Longest Substring Analysis
  • Find the length of the longest substring
  • Identify the longest substring
  • Generate a list of all unique substrings
Search History

View and manage your complete history of analyzed strings

Usage

Enter your string in the input field and click "Analyze" to get instant results.

Tree Visualizer

Interactive binary tree visualization and analysis tool.

Features

  • Add individual nodes to the tree
  • Generate a complete binary tree based on specified depth
  • Delete entire tree or specific nodes

  • Find maximum sum path from leaf to node
  • Calculate maximum sum path from any node to any other node
  • Visual highlighting of optimal paths

Quick Start Guide

  1. Click the to manually configure nodes
  2. Or use "Generate Tree" to create a complete tree
  3. Click The Max Leaf Path button to calculate the Max sum path from leaf to node
  4. Click The Max Node Path button to calculate the Max sum path from a node to any node
